## What is CoVound?
CoVound is an **Active AI Security Firewall** designed to intercept social engineering attacks at the moment of intent—before the first digit is dialed.

It functions as a "Digital ER Doctor," triage-ing incoming search data, diagnosing anomalies, and prescribing verified truths with a calm, educational authority.

### The Ecosystem
1.  **The Brain (Vounder Registry & Anomaly Engine):** A cryptographically verified database of official contact points. The Anomaly Engine uses the **Google Gemini API** to actively hunt the web for SEO-poisoned results, flagging them in the registry before they can find their next victim.
2.  **The Shield (CoVounder MCP Server):** A Model Context Protocol server that forces AI assistants (Gemini, Claude) to query our verified registry instead of relying on generic web hallucinations. It ensures the AI speaks with the "Dr. Gia" persona—calm, empathetic, and strictly factual.
3.  **The Triage (Voundism Browser Extension):** A frontline Chrome extension that scans search results in real-time. If it detects a known scam number or an unverified entity, it visually redacts the threat and replaces it with a verified path to safety.
